> ### How to execute jar
> **Server**: launched if no argument is given<br>
> **Client**: launched with either "CLI" or "GUI" given as argument<br>
> > *java -jar MyShelfie.jar* <br>
>
> > *java -jar MyShelfie.jar CLI*<br>
> > *java -jar MyShelfie.jar GUI*
